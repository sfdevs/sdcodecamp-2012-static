South Dakota Code Camp

is hosted by the Sioux Falls Developers Group and South Dakota .Net User Group with the sponsorship and support from many area businesses. Most of you may be wondering "What's a Code Camp?" Here is the basic idea.

A Code Camp IS:

Community Driven
Free to attendees
Relevant to community members
NEVER during work hours (usually on Saturdays)

A Code Camp IS NOT:

Product Demonstration/Sales
Corporate Marketing
